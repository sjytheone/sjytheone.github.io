城市轨道交通出行诱导系统

发表于 2017-01-25

Tsmart for Android

一款针对Android设计的轨道交通查询系统 从fir下载体验

General

  • Author : sjy
  • Version : 1.0.14
  • Compatibility : Android 5.0 +

Features

General features :

  • 线路预览
  • 车次预览
  • 车站查询
  • 车次到站时间预览
  • 车次时刻表查询
  • 基于车次时刻表的轨道线路规划
  • 预览车次方向，车辆拥挤度查看
  • 多种颜色主题可选

Posts features :

  • 快速Json数据解析
  • 基于Android5.0+的svg图标绘制
  • 简洁优雅的Material Design
  • 超大图片加载，移动及响应
  • 先进的线路规划算法..
  • 车次拥挤度查看

Documentation

  • 加载显示超大图片:Subsampling Scale Image View
  • 快速Json解析:Fast Json
  • 路径规划展示控件:BottomSheet
